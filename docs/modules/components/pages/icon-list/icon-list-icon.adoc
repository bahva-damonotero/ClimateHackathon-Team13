= Icon List Icon

The icon for an xref:icon-list/icon-list-item.adoc[Icon List Item].

See also: xref:icon-list/index.adoc[Icon List].

== Props

In addition to the below props, any HTML attributes (such as `id`, `title`, etc.) can be used and will be passed through to the opening `<div>` tag.

[grid="rows"]
|===
| *Name* | *Type* | *Default* | *Description*
include::../shared/shared.adoc[tag=className]
| color | `String` | | Color for the icon. For options, see https://designsystem.digital.gov/utilities/color/[here].
| identifier | `String` | | The icon to use. For options, see https://designsystem.digital.gov/components/icon/[here].
| sprite | `String` | | The path to USWDS's `sprite.svg` or another SVG sprite.
|===
